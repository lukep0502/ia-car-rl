# IA-CAR 🚗🤖

Top-down racing simulator built with Pygame.
Project focused on developing and training reinforcement learning agents.

## Phase 1 - Core Engine
- Circular track
- Lap counting system
- Direction enforcement (clockwise)
- Finish line validation
- Timer system
- Restart system

## Next Phases
- Custom track shapes
- Checkpoint system
- Sensor system
- Gymnasium integration
- Reinforcement Learning training
